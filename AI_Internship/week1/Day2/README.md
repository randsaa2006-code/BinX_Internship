## Topics Covered 📚

### 1. Core Python Data Types

Explored the main Python data structures used in programming and data analysis:

- **int:** Represents integer values.
- **float:** Represents decimal numbers.
- **str:** Represents text values.
- **bool:** Represents True or False values.
- **list:** Ordered and mutable collection of elements.
- **tuple:** Ordered and immutable collection of elements.
- **dict:** Stores data as key-value pairs.
- **set:** Stores unique unordered values.

Practiced creating and manipulating different data types and understanding when to use each one.

---

### 2. Control Flow

Learned how to control the execution of Python programs using:

Conditional statements:

```python
if
elif
else
```

Loops:

```python
for
while
```

Practiced applying conditions and iterations to solve programming problems and automate repetitive tasks.

Understanding control flow is important for implementing custom logic in data processing and cleaning tasks.

---

### 3. Functions

Learned how to create reusable blocks of code using functions.

Practiced:

- Defining functions.
- Using parameters.
- Returning values.
- Writing clear function documentation using docstrings.

Example:

```python
def function_name(parameters):
    """
    Function description
    """
    return result
```

Functions help create cleaner, reusable, and maintainable code for data workflows.

---

### 4. List Comprehensions

Learned how to create lists in a more concise and Pythonic way.

Practiced replacing traditional loops with list comprehensions.

Examples:

```python
[x**2 for x in range(10)]
```

and:

```python
[x for x in range(20) if x % 2 == 0]
```

List comprehensions improve code readability and are commonly used in Data Science tasks.

---

### 5. Object-Oriented Programming Basics

Explored the fundamentals of Object-Oriented Programming (OOP).

Learned:

- Creating classes.
- Defining attributes.
- Creating methods.
- Initializing objects using `__init__()`.

Example concepts:

```python
class ClassName:
    def __init__(self):
        pass
```

Understanding OOP is important because many AI and Machine Learning libraries, such as Scikit-learn and PyTorch, are built using classes and objects.
