## Day 2: Python Fundamentals for Data Science

In this day, I practiced the core Python concepts required for AI and Data Science workflows.

Topics covered:
- Python built-in data types (int, float, string, boolean, list, tuple, dictionary, and set)
- Control flow using conditional statements and loops
- Writing reusable functions with parameters, return values, and docstrings
- List comprehensions for efficient list creation
- Basics of Object-Oriented Programming (classes, attributes, and methods)

Hands-on practice included building custom functions, applying list comprehensions, and creating simple classes to strengthen Python programming skills.
