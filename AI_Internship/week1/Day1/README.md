## Day 1: AI/ML Environment Setup

In the first day, I prepared a reproducible Python environment for AI and Machine Learning development.

## Topics Covered

- Creating and managing Python virtual environments using venv.
- Installing and managing packages using pip.
- Setting up Jupyter Notebook for interactive development.
- Verifying installed libraries and their versions.
- Managing project dependencies using requirements.txt.
- Using Git and GitHub for version control.

## Hands-on Practice

- Created an isolated Python environment for the project.
- Installed essential Data Science libraries:
  - NumPy
  - Pandas
  - Matplotlib
  - Jupyter
- Verified the environment setup by checking library versions.
- Initialized a Git repository and pushed the project files to GitHub.

This setup provides a reproducible foundation for future AI and Machine Learning experiments.
