## Day 3 - NumPy: Numerical Computing

Today, I learned the fundamentals of NumPy, the core library for numerical computing in Python.

Topics covered:
- Understanding NumPy arrays (ndarray) and why they are important in Data Science and AI.
- Creating arrays using different methods such as array(), zeros(), ones(), arange(), linspace(), and random().
- Understanding array attributes including shape, dtype, ndim, and size.
- Reshaping arrays and changing their dimensions without modifying the data.
- Applying indexing and slicing techniques on one-dimensional and two-dimensional arrays.
- Performing vectorized operations for faster numerical computations without explicit loops.
- Using Boolean Masking for filtering data based on conditions.
- Understanding Broadcasting and how NumPy performs operations between arrays of compatible shapes.

Practical work:
- Created and manipulated NumPy arrays.
- Applied slicing, masking, vectorization, and broadcasting through hands-on exercises.
